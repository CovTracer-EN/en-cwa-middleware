# en-cwa-middleware
"Google Exposure Notification Server" - "Corona Warn App Server" middleware

## Production
### Run
```console
./en-cwa
```

## Development

### Prerequisites
- [Golang](https://golang.org/dl/) >= 1.14
- [UPX](https://upx.github.io/) (optional for binary optimization)

### Run
```console
make
```

### Build
```console
make build
```
